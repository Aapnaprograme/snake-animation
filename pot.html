Programming Languages

Programming languages are used to communicate with computers and build software.

  • HTML

    HTML (HyperText Markup Language) is used to create the structure of web pages. It defines elements like headings, paragraphs, images, and links.

  • CSS

    CSS (Cascading Style Sheets) is used to style web pages. It controls layout, colors, fonts, animations, and responsiveness.

  • JavaScript

    JavaScript adds interactivity to websites. It is used for dynamic content, animations, form validation, and logic.

  • Python

    Python is a powerful and beginner-friendly programming language. It is widely used in web development, data science, AI, and automation.

Frameworks, Platforms & Libraries

Frameworks, libraries, and platforms help developers build scalable, fast, and maintainable applications.

Frameworks

React

React is a JavaScript framework used to build fast and interactive user interfaces, especially single-page applications.

Angular

Angular is a TypeScript-based framework used for building large-scale and enterprise-level web applications.

Vue

Vue is a lightweight JavaScript framework that is easy to learn and used for building modern web interfaces.

Django

Django is a Python web framework that helps in building secure, scalable, and backend-driven web applications.

Spring Boot

Spring Boot is a Java-based framework used to create powerful backend services and REST APIs.

Libraries

jQuery

jQuery is a JavaScript library that simplifies DOM manipulation, animations, and event handling.

NumPy

NumPy is a Python library used for numerical computing and working with large arrays and matrices.

Pandas

Pandas is a Python library used for data analysis, data manipulation, and data cleaning.

TensorFlow

TensorFlow is a machine learning library used to build deep learning and AI models.

Platforms

Node.js

Node.js is a JavaScript runtime that allows developers to run JavaScript on the server side.

.NET

.NET is a Microsoft platform used to build web, desktop, and cloud-based applications.

Databases & ORM

Databases are used to store, organize, and manage application data efficiently. ORM tools help developers interact with databases using programming languages.

Databases

MySQL

A popular relational database used for web applications.

PostgreSQL

An advanced open-source relational database with high performance.

MongoDB

A NoSQL database used to store data in JSON-like documents.

SQLite

A lightweight database used in mobile and small applications.

ORM Tools

Hibernate

A Java ORM framework used to map Java objects to database tables.

Sequelize

A Node.js ORM for SQL databases like MySQL and PostgreSQL.

SQLAlchemy

A Python ORM that provides powerful database abstraction.

Prisma

A modern ORM that simplifies database access for JavaScript apps.

Machine Learning & Deep Learning

Machine Learning (ML) and Deep Learning (DL) help computers learn from data and make decisions without being explicitly programmed.

What is Machine Learning?

Machine Learning is a branch of Artificial Intelligence that focuses on building systems which learn patterns from data and improve automatically with experience.

  • Supervised Learning
  • Unsupervised Learning
  • Reinforcement Learning

What is Deep Learning?

Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers to analyze complex data such as images, audio, and text.

  • Neural Networks
  • Convolutional Neural Networks (CNN)
  • Recurrent Neural Networks (RNN)

Popular ML & DL Tools

These tools and libraries are widely used to build intelligent applications and AI models.

  • TensorFlow
  • PyTorch
  • Scikit-learn
  • Keras

CI/CD & Version Control

CI/CD and Version Control help developers manage code, collaborate easily, and automate building, testing, and deploying applications.

What is Version Control?

Version Control is a system that tracks changes in source code. It allows multiple developers to work together and maintain different versions of the project safely.

  • Track code changes
  • Collaborate with teams
  • Rollback to older versions

Popular Version Control Tools

These tools are widely used to host, manage, and review source code in modern software development.

  • Git
  • GitHub
  • GitLab

What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. It automates the process of building, testing, and deploying applications.

  • Continuous Integration (CI)
  • Continuous Deployment (CD)
  • Automated testing

CI/CD Tools

These tools help automate workflows and ensure faster and more reliable software delivery.

  • GitHub Actions
  • Jenkins
  • GitLab CI/CD

Testing

Testing ensures software quality and reliability.

  • Unit Testing
  • Integration Testing
  • Automation Testing
  • Selenium
  • JUnit
  • PyTest
